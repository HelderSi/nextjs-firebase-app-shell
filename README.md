### NEXTJS WITH FIREBASE APP SHELL

An App starter project that has NextJS with Firebase set up beforehand. Go ahead and clone it to start your project quickly.

## Dependencies

- NextJs
- Firebase
- Flowbite
- Next Auth
- Firebae Admin
- Next Auth FirestorAdapter
